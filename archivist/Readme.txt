# GENERAL INFORMATION #

Archivist is a program for "Magic the Gathering" developed by Richard Garfield.

Archivist allows you to create Magic Decks for Mindless Automaton and
Apprentice. You can manage your private card collection, too.
All card information are read from E-League`s very popular Cardinfo Files. You
can add new editions by simply replacing the Cardinfo and Expan files.

You can download card images and view them in Archivist.


# HOW TO VIEW CARD IMAGES #

The next version will allow you to download images from the Wizards page.
Currently you can download images from http://www.wizards.com/magic/welcome.asp
and put them into "cardimg". Spaces, "'" and "," are removed.
Zur's Weirding => ZursWeirding
Black Lotus => BlackLotus


# UPDATE CARD INFORMATION #
Download the latest CardInfo.dat and Expan.dat from "www.e-league.com".
(Usually these are part of Apprentice)
